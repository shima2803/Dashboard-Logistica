# 📊 Análise de Dados de Logística

Dashboard desenvolvido no **Power BI** para análise de dados logísticos de uma operação de entregas, com foco em performance por região, canal, equipe e status de entrega.

![Dashboard de Logística](./imagem/dashboard.png)

---

## 🎯 Sobre o Projeto

Este projeto faz parte do **Lab 06 - Logística** e tem como objetivo aplicar conceitos de análise de dados e visualização para extrair insights sobre o desempenho de uma operação logística, identificando pontos de melhoria e oportunidades de otimização.

---

## 📈 KPIs e Visualizações

O dashboard apresenta os seguintes indicadores e gráficos:

### Indicadores Principais
- **Total de Entregas:** 53.770 mil
- **Entregas no Prazo:** 47 mil

### Visualizações Disponíveis
- **Total de Entrega No Prazo Por Canal Da Entrega** — Gráfico de área mostrando a distribuição por canal
- **Total de Entrega por Mês** — Linha temporal com a evolução mensal das entregas
- **Percentual de Entrega Por Equipe** — Comparativo entre as regiões (Norte, Sudeste, Nordeste, Sul, etc.)
- **Percentual de Entregas Por Status** — Distribuição entre Antecipado, No Prazo e Atrasado
- **Ranking de Vendedores** — Tabela com top vendedores e suas avaliações
- **Cidades com Maior Atraso** — Tabela detalhada de entregas com atraso por cidade

### Filtros
- Filtro por ano: **2019** e **2020**

---

## 🔍 Principais Insights

- A região **Norte** lidera em volume de entregas (14,8 mil), seguida pelo **Sudeste** (13,4 mil)
- **70,71%** das entregas são realizadas de forma **antecipada**
- Apenas **16,31%** das entregas ocorrem exatamente no prazo
- **12,98%** das entregas apresentam atraso
- Há uma queda relevante no volume de entregas a partir de **setembro**

---

## 🛠️ Tecnologias Utilizadas

- **Microsoft Power BI** — Desenvolvimento do dashboard
- **Power Query** — Tratamento e modelagem dos dados
- **DAX** — Criação de medidas e cálculos
